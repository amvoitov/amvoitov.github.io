## Коротко о проекте
**Универсальная самоходная платформа с модулем опрыскивания** — роботизированная гусеничная платформа с гибридным приводом для работы в **виноградниках и садах**, с управлением **вручную / по маршруту на карте / автопилотом**.

## Заказчик и пользователи
- **Заказчик/владелец:** частный заказчик (NDA)
- **Пользователи:** механизатор / агроном / инженер

## Боль (почему проект запустили)
1) Нехватка людей / высокая стоимость работ по опрыскиванию  
2) Риски для людей (химия) и требования безопасности  
3) Нестабильное качество обработки (перекрытия/пропуски/неравномерность)  
4) Сложные условия эксплуатации: узкие междурядья, развороты, уклоны, грунт  
5) Высокая стоимость владения и простои техники

## Цели
1) Снизить трудозатраты оператора / частично заменить ручной труд  
2) Повысить качество внесения (уменьшить перекрытия/пропуски)  
3) Повысить безопасность (минимум контакта людей с химией)  
4) Увеличить производительность (га/час) при сохранении качества  
5) Снизить стоимость владения и простои

## Метрики (из текущих требований/гипотез)
- **Производительность:** 1 га/час  
- **Автономность:** 10 часов  
- **Скорость движения:** 1–10 км/ч  
- **Точность дозирования / перекрытия/пропуски / надёжность:** метрики в % — пока не зафиксированы

## Состав системы
- **Шасси/платформа:** гусеничное шасси  
- **Привод:** гибридный  
- **Модуль опрыскивания:** бак, насос, штанга/форсунки, секции, управление нормой  
- **Навигация (GNSS/RTK, NMEA):**  В контуре навигации использовались данные GNSS/RTK для устойчивого движения по маршруту и работы автопилота.
      Обрабатывались параметры движения и качества позиционирования: координаты, скорость, курс, трек,
      HDOP/VDOP, RTK Fix/Float<. Поддерживались варианты получения данных в формате NMEA.
- **В плане разработки (сенсоры/функции):**
  - камеры — обнаружение препятствий и рядов  
  - ультразвук — поддержка движения/контроля ряда  
  - определение расхода химии  
  - датчик уровня жидкости + оценка “хватит ли химии на проход по ряду”
- **Управление (в испытаниях работает):** ручной / маршрут на карте / автопилот  
- **Интерфейсы:** web / планшет / мобильное

## Моя роль и вклад
**Роль:** PM / PO / аналитик / архитектор / техлид  
**Делал:** требования, план/сроки, бюджет, команда/подрядчики, архитектура, испытания, внедрение, документация, финмодель и юнит-экономика, анализ рынка и конкурентов, питчи перед инвесторами, CustDev, техническая документация, полная документация проекта, участие в выставке **ЮгАгро 2025**.
